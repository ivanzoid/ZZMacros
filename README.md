## Overview

A set of often used macros.

## Usage

`#import "ZZMacros.h"`
See particular `ZZMacro`\* file for more details.

## Installation

ZZMacros is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

    pod "ZZMacros"
